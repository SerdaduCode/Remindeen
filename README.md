# Remindeen
## Prayer reminders and daily inspiration while you browse
Remindeen is a browser extension specifically designed to be a
prayer reminder and source of inspiration for Muslims in carrying
out their daily activities.

### V.1.0.0
- Initial release with basic prayer time reminders and features a user-friendly interface.
### V.1.1.0
- Add some UI improvements and bug fixes.
### V.1.2.0
- Another UI improvements and bug fixes.
### V.1.3.0
- Add notification for shalat reminders
- Add support for English language for menu settings
- Add side panel with support section, an easy way to support the development of the extension.
- Add support for dark mode, providing users with a more comfortable browsing experience in low-light environments and enhancing the overall aesthetics of the extension.
- Browsing Tracker to record the time spent on each website, it saves local in user browser, and reset everyday automatically.
### Upcoming Feature
- AI Powered Browsing Tracker that classifies the websites into categories such as work, entertainment, social media, etc., providing users with insights into how they spend their time online and helping them manage their digital well-being more effectively.
- Charts and statistics to visualize the time spent on different categories of websites, giving users a clear overview of their browsing patterns and helping them identify areas where they can improve their time management.
- Add Avatar named "Hanif" animation to enhance the user experience and make the extension more engaging and visually appealing.
